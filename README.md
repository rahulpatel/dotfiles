# Dotfiles

Run `xcode-select --install` then run the following...

```bash
cd /tmp
curl -LO https://github.com/rahulpatel/dotfiles/archive/master.zip
unzip master.zip
mv ./dotfiles-master ~/.dotfiles
cd ~/.dotfiles
script/bootstrap
```
